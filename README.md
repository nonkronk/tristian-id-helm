## Installation


```bash
  helm repo add nonkronk https://charts.tristian.id
```

```bash
  helm repo update
```

```bash
  kubectl label nodes <your-node-name> app-deployment=true
```

```bash
  helm install pwa nonkronk/tristian-id
```
